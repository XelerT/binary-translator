\doxysubsection{Detailed Description}
Contains information about code for JIT compilation. 

It contains pointer to buffer with encoded x86 commands, capacity and size of buffer, exec\+\_\+memory2use is pointer to the \char`\"{}data\char`\"{} section, offsets for print and scan functions.
